> # 警告！当前版本为 Alpha 版本，请勿在生产环境使用
>
> **⚠️ 警告**：当前版本为 **Alpha 版本**，**请勿在生产环境使用**。部分功能可能不稳定或不支持，仅剩余少量功能未移植，仅供用户体验。使用中如遇问题，欢迎前往 GitHub Issues 或 QQ 频道 反馈！

v2.0 - Koharu（小鸟游星野） Alpha 4

## 🚀 主要更新

- 新增 **语音功能开关**，可关闭语音播报功能

## 💡 功能优化

- 优化 **浮窗隐藏功能**，解决浮窗隐藏不完全问题
- 优化 **浮窗美观度**，提升浮窗视觉效果
- 优化 **导入名单功能**，提高加载速度
- 优化 **更新下载**，解决下载超时导致失败问题

## 🐛 修复问题

- 修复 **资源目录问题**，解决软件无法打开的问题
- 修复 **剩余名单显示**，解决显示失败问题
- 修复 **构建问题**，解决软件无法打开的问题

## 🔧 其它变更

- **开发计划**，重新梳理并明确后续开发方向：**自定义抽**、**即抽**、**URL功能**、**IPC功能**、**安全功能优化** 等将逐步完善

---

💝 **感谢所有贡献者为 SecRandom 项目付出的努力！**
我们坚持 **便捷、迅速、公平、透明** 的理念，让校园及更多场景的点名、抽奖、决策更安心。欢迎大家试用 Alpha 版，并踊跃反馈问题与建议，一起打磨更好的 SecRandom!
